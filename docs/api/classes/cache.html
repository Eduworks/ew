cache

Caches a result, and fetches it automatically (without executing the code in obj) if it is in cache.
Cache, by default, persists over the web service request.

rs2: result = obj.cache(name="unique name");
LevrJS: result = cache.call(this,obj,"unique name");

Name Type Attribute Description

Cruncher or Function that will return an object to cache.

Unique name used to store and retrieve the cached object.

Removes the cached value using the name. 'obj' can be null. May be combined with 'global'.

  • optional
  • default: false

Add to global cache, persists over web service requests.

  • optional
  • default: false

Don't deal with the cache, but use the cache locking mechanism to prevent recurrent entrance.

  • optional
  • default: false

Clears all caches in LEVR.
